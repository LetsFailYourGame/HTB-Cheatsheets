### Nmap Scripting Engine
* Nmap Scripting Engine (`NSE`) provides possibility to create scripts in Lua
* 14 different categories like `auth`, `brute`, `discovery` etc.
* Default `-sC` executes common scripts
* Use `--script <category>` to specify category
* Use `--script <script-name>,<script-name>,..` to define scripts

```sh
$ sudo nmap 10.129.2.28 -p 25 --script banner,smtp-commands

Starting Nmap 7.80 ( https://nmap.org ) at 2020-06-16 23:21 CEST
```

* Also, we can use aggressive scanning with `-A`
	* Scans target with service detection`-sV`, OS detection `-O`, traceroute `--traceroute`, and default NSE scripts `-sC`
